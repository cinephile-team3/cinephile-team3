# 🎬 Cinephile — Precision Engineered Streaming

> Developed as a learning project within the FemCoders Bootcamp.

## 📋 Description

Cinephile is a premium streaming platform landing page built with React. The project includes three fully responsive pages — Home, Pricing, and Contact — with navigation between them and a design faithful to the provided mockups.

## 🛠️ Tech Stack

- **React** + **Vite** — component-based UI framework
- **React Router DOM** — client-side navigation
- **Sass** — variables, mixins, and BEM methodology
- **Bootstrap Icons** — iconography
- **Mobile First** — responsive design for all devices

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/cinephile-team3/cinephile-team3.git

# Navigate to the project folder
cd cinephile-team3

# Install dependencies
npm install

# Start the development server
npm run dev
```
